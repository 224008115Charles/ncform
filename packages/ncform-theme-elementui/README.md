# How to dev

Step 1: prepare the dependency nc libraries
```sh
cd ../../ && npm run build
```

Step 2: start development
```sh
npm run dev
```

[可选]配合以下可使用依赖的`ncform-common`的最新代码
```
cd ../ncform-common
npm run watch-build
```

[可选]配合以下可使用依赖的`ncform`的最新代码
```
cd ../ncform
npm run watch-build
```
